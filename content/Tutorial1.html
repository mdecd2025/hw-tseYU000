Tutorial1:Your First Simulation in Webots

當修改 Webots world並希望保存時，至關重要的是首先暫停模擬並重新加載到其初始狀態，即主工具列上的虛擬時間計數器應顯示 0:00:00:000。否則每次儲存時，每個 3D 物件的位置都會累積錯誤。對世界的任何修改都應按照以下順序進行：暫停、重置、修改和保存模擬。

  • WorldInfo：設定模擬的各種參數
  • Viewpoint : 定義主視點相機參數
  • TexturedBackground：定義場景的背景
    • 左鍵點擊選擇Add new 選擇Base nodes 可新增自己需要的實體或形狀。
  • TexturedBackgroundLight：定義與上述背景相關的光
  • RectangleArena (Derived from Solid)：定義您在該場景看到的物件
    • 選擇floorTileSize、wallHeight可更改場地大小與牆壁高度。

新增物件:

雙擊RectangleArena 關閉他，再選取他左鍵點擊會顯示Add nwe 。選擇 PROTO nodes (Webots Projects) / objects / factory / containers / WoodenBox (Solid)，競技場中央會出現一個箱子，在場景樹中雙擊它以打開字段。

  • size : 可更改物件大小。
  • translation : 更改數值改變旋轉方向，或直接選取物件拉取要選轉的軸進行更改。
  • 更改完後記得進行儲存。

建立新控制器

Webots控制器可以用以下程式語言編寫：C、C++、Java、Python、MATLAB等。 C、C++和Java控制器需要先進行編譯才能作為機器人控制器運作
